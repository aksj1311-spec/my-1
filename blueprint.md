# **안수녀님 명언봇 (Sister Ahn Quote Bot)**

## **개요 (Overview)**
이 프로젝트는 안수녀님의 따뜻하고 애정 어린 잔소리와 명언을 픽셀 게임 감성으로 전달하는 웹 애플리케이션입니다. 사용자 이름을 입력받아 명언에 적용하며, 90년대 레트로 게임의 UI 스타일(그레이톤, 픽셀 폰트)을 지향합니다.

## **상세 설계 및 기능 (Detailed Design & Features)**

### **1. 디자인 스타일 (Design Style)**
*   **픽셀 아트 감성:** 모든 UI 요소(버튼, 입력창, 테두리)를 픽셀 게임 느낌으로 구현. `box-shadow`를 활용한 픽셀 보더 효과 적용.
*   **그레이톤 팔레트:** 그라데이션을 배제하고 무채색 계열(화이트, 라이트 그레이, 다크 그레이, 블랙)만 사용하여 깔끔하고 귀여운 느낌 전달.
*   **타이포그래피:** Google Fonts의 'Silkscreen' 또는 'Press Start 2P'와 같은 픽셀 폰트 사용 (한글의 경우 둥근모꼴류의 픽셀 폰트 사용).

### **2. 주요 기능 (Core Features)**
*   **사용자 이름 입력:** 이름을 입력하는 칸을 배치하여, 명언 출력 시 "[이름]야, ..." 와 같이 친근하게 이름을 불러주는 효과.
*   **명언 랜덤 출력:** 안수녀님의 실제 말씀을 기반으로 다듬어진 명언 리스트 중 하나를 랜덤으로 화면에 표시.
*   **말투 구현:** 다정하지만 약간의 '잔소리'가 섞인 안수녀님만의 독특한 문체 재현.

### **3. 기술 스택 (Tech Stack)**
*   **HTML5 / CSS3:** 프레임워크 없이 순수 웹 표준 기술 사용.
*   **JavaScript (ES6+):** 명언 데이터 관리 및 랜덤 출력 로직 구현.
*   **Web Components:** UI 재사용성을 위해 필요 시 커스텀 엘리먼트 활용.

## **구현 계획 (Current Plan)**

### **1단계: HTML 구조 설계**
*   메인 컨테이너, 이름 입력창, 명언 출력 버튼, 텍스트 출력 영역 구성.

### **2단계: CSS 스타일링**
*   픽셀 보더 스타일링 및 그레이톤 색상 테마 적용.
*   반응형 레이아웃 구현.

### **3단계: JavaScript 로직 구현**
*   안수녀님 명언 데이터베이스 구축 (10개 이상의 문항).
*   이름 입력값 연동 및 명언 생성 함수 작성.
*   클릭 이벤트 핸들러 연결.

### **4단계: 마무리 및 검증**
*   브라우저 테스트 및 오류 수정.
